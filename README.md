# Downloads - Browser Idle Tycoon

[![Website](https://img.shields.io/badge/website-online-success?style=for-the-badge&logo=github)](https://wattox00.github.io/game)
[![License](https://img.shields.io/badge/license-GPLv3-blue?style=for-the-badge)](LICENSE)

A small browser-based incremental/tycoon game where you download files, manage limited storage, upgrade your internet, and sell prized data to grow your fortune. Progress from junk files to cosmic secrets and master the black market.

**Live demo:** https://wattox00.github.io/game


**Quick Start**
- **Money:** Earn by selling downloaded files.
- **Downloads:** Click items in the Downloads view to download files to storage.
- **Storage:** Manage capacity; upgrade to hold larger files.
- **Internet Speed:** Faster speeds reduce download time.
- **Upgrades & Systems:** Unlock the Black Market, Printer (copying), Autoseller, Autobuyer, and bulk downloads as you progress.

**Gameplay Overview**

- Start with a tiny Floppy Disk and a slow internet connection.
- Download small files (e.g., "Junk") to earn your first money.
- Upgrade internet speed and storage to access and hold larger, more valuable files.
- Use the Black Market to temporarily boost selling prices.
- Unlock the Printer to duplicate items, then automate selling or reprinting.
- Continue upgrading until you can download the ultimate prize: "The BIG SECRET".

Features

- Progressive item tiers from small text files up to "The BIG SECRET" (see [script.js](script.js) for the full list).
- Multiple storage types (Floppy → CD-ROM → DVD → HDD → SSD → NVMe → Neural Vault → Galaxy Archive Disk).
- Internet speed upgrades with increasing costs and throughput.
- Black Market boosts: temporary multipliers for specific item types.
- Printer system with autoseller/autobuyer and double-print upgrades.
- Bulk download options after certain unlocks (1x, 5x, 10x, 25x).
- Clean, responsive UI powered by [index.html](index.html), [style.css](style.css), and [script.js](script.js).

Files of interest
